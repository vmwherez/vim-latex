# vim-latex

Compose LaTeX with Vim

`sudo apt install textlive-full latexmk`

`pdflatex`

`https://www.tug.org/mactex/`

`https://www.tug.org/mactex/morepackages.html`

On MacOS you can just use the Preview application.

```
$ pdflatex file.tex
$ open file.pdf
```
